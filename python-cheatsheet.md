### Basics

##### Arithmetic Operations
```python
>>> 5 + 2   # Addition
7
```
```python
>>> 4 - 3   # Subtraction
1
```
```python
>>> 4 * 4   # Multiplication
16
```
```python
>>> 10 / 2  # Division
5.0         # Return float value

>>> 10 // 2 # Division, but return integer value
5
```
```python
>>> 10 % 2 # Modulas(Remainder)
0
```
